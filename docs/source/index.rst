欢迎与我一同来读《圣经》
======================

开这个系列篇章的本意：

**阅读圣经**。透过阅读，理解圣经里传达的信息，前后一致无矛盾的理解下面几件事：

1.你是人，不只拥有肉体躯壳，你的本尊是个能夠永活的灵魂。
2.你这代人（六千年以来）生于灵性的低谷，生命力低下的状态。
3.你需要外力拉一把，以救赎你脱离低谷，到达永生的喜悦世界生活。

圣经记录的年代前后长达数千年，厚厚一本要断章取义片面理解其实更容易的。但这里不打算走片面的那条路。
敢做这个尝试是因为：我读过整本，不只一次。另外，我有一位好老师，他非常擅长见证耶和华神，以及耶稣所做过，发生在地球上的各种事。

至于信仰，我无意知道你信或不信，你信也好不信也罢。既然你来了，那么就随我一同来阅读理解，圣经背后的那位作者—耶和华—为什么拚命的希望地球人相信他。

我也不想辩经。如果我漏掉了新旧约圣经里某段的记载，以致于说出了自打嘴巴的矛盾理解或说明，请你把我漏掉的经文翻出来指教指教。至于教派各自的理解差异，请回到您各自教派去争辩。那是你们在圣经记载以外归纳出来的教理教义假设，我不是你教父不是你老师，我没有为你证明或维护你教义的义务。

阅读重点在于：因为圣经的背后作者，是一位、同一位有人格，从古到今说话一致行动目标一贯的存在者。既然圣经作者耶和华神要人相信他，那么他说话就要前后一贯不能颠三倒四，即使跨越了几千年，也不能否定自己以前讲过的。

每一篇都是入门帖，不假定你读过圣经，不假定你认识至高神，不假定你认识基督教各派及犹太教。如果前情提要篇幅过长，我就放个页面连结。
